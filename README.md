# potato-Pepper-Disease-Detection
This application categorizes diseases of pepper and potato plants 

The Algorithm and web app was built in Python using the following libraries:
* streamlit
* pandas
* numpy
* tensorflow
* keras 
* PLT
* cv2
* google colab 
* google Drive 
* ngrok
